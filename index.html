Information

Accepted goals are the following:
  • GET [OBJECT] [ROOM]: Get object from specified room
  • GET [OBJECT]: Get object from any room
  • PUT [OBJECT] [ROOM]: Put object in specified room
  • PUT [OBJECT]: Put object in any room (can be used to drop held object in current room)
  • GOTO [ROOM] [OBJECT]: Go to specified room holding object
  • GOTO [ROOM]: Go to specified room

[OBJECT] can be either ORB, KEY or NOTHING.
[ROOM] can be a room name, found in top-right corner of rooms on map.

Map legend:
  • O: Orb
  • K: Key
  • R: Robot
  • DD: The path between these rooms is blocked by a door, impeding progress unless you have a key. Can be forced open, but doing this costs more energy
  • ' ': A blank space between rooms means there is a hallway connecting them, allowing unimpeded access
Action costs:
  • Pickup object: 0.25
  • Drop object: 0.25
  • Move through hallway: 1
  • Open door with key: 1
  • Force open door: 10
